# Food Store – Sistema de Gestión de Pedidos

Trabajo Práctico Integrador – Programación 2  
Tecnicatura Universitaria en Programación (TUPaD) – UTN

## Descripción
Sistema de consola desarrollado en Java para la gestión de productos, categorías, usuarios y pedidos de un negocio de comidas. Implementado con POO, colecciones en memoria y manejo de excepciones propias.

## Cómo ejecutar el proyecto

### Requisitos
- JDK 21 o superior
- IDE compatible con Java

### Pasos
1. Clonar o descomprimir el proyecto
2. Abrir el proyecto en NetBeans
3. Ejecutar la clase `Main.java` ubicada en `src/integrador_prog2/`
